iMessage

An all-encompassing texting experience


The design challenge was to redesign Apple's iMessaging ecosystem, integrating the messaging interface, contacts, and texting settings all in one place.

view design 
Skills Applied:
  • Competitor Analysis
  • Interface Design
  • Figma Prototyping

Interface Design

Goals
  • An interface that was minimal and intuitive to use.
  • Integrates communication features all into one place.
  • Encourages engagement between users.

Messages Page

This project was a demonstration of my initial depth in learning Figma tools. Auto-Layout was the most efficient tool I learned so far. The tool enabled me to duplicate components at a much more efficient rate, and made re-editing frames far less tedious.

Headings on top help the user easily locate where they are. The heading and menu bar are fixed in place for easy access when the page scrolls down.

The colors green, yellow, red, and no-color indicate whether the user is online, idle, in do not disturb, or offline, respectively. These colors are chosen based on its commonality among other platforms.

Chat Page

Overall, the messaging experience of the re-design strongly resembles iMessage. However, additional changes like the settings button in the heading allows the user to edit the recipient's contact information.

Figma's auto layout tool was used again in the messaging page, especially the chat log, where the text bubble will automatically adjust to the text being added.

Contact Page

One of the most distinguished feature I incorporated to the iMessage re-design is adding the contacts page and profile page to the app. This makes searching and adding new contacts more convenient as it can be completed within the same app.

The right side vertical bar lists the first alphabetical letter of the contact lists. The top alphabet category on the user view screen will be bolden in the list to indicate what alphabet they are on. For example, viewing contacts that start with the letter H will bolden the "h" letter in the list.

Profile Page

Furthermore, rather than having the iMessage settings in the Settings app, it makes more sense and convenience to also have it in the iMessage app.

Clicking on the status button will open up a status changer option using an overlay opening interaction in Figma.

Settings Page

In the settings page, only the most essential settings are kept while less frequently changed settings can be adjusted in the iOS Settings app. Some of the settings are inspired from apps like WeChat, Facebook Messenger, and iMessage in Settings.
